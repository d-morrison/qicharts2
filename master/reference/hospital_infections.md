# Hospital acquired infections

A dataset containing the number of hospital acquired bacteremia,
Clostridium difficile infections, and urinary tract infections in six
hospitals in the Capital Region of Denmark 2015-2016.

## Usage

``` r
hospital_infections
```

## Format

A data frame with 432 rows and 5 variables:

- hospital Abbreviated hospital name.

- infection Type of infection. BAC: Bacteremia, CDI: Clostridium
  difficile infection. UTI: Urinary tract infection.

- month First day of month.

- n Number of cases.

- days Number of risk days. A risk day is a patient day without
  infection.

## Source

www.esundhed.dk (Capital Region of Denmark).
