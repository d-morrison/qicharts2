# NHS accidents

The number of attendances to major accident and emergency hospital
departments in the NHS that were seen within 4 hours of arrival over
twenty weeks.

## Usage

``` r
nhs_accidents
```

## Format

A data frame with 20 rows and 3 variables:

- i Week number.

- r Attendances seen within 4 hours.

- n Total number of attendances.
