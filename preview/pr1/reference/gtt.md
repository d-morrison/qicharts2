# Patient harm identified using the Global Trigger Tool

A dataset with data on adverse events during hospitalisation found by
the Global Trigger Tool.

## Usage

``` r
gtt
```

## Format

A data frame with 340 rows and 11 variables:

- admission_id Admission ID.

- admission_dte Date of admission.

- discharge_dte Date of discharge.

- month Month of discharge.

- days Duration of hospital stay in days.

- harms Number of adverse events.

- E-I Type of adverse event by severity category. E-F: Temporary harm;
  G-H: Permanent harm; I: Fatal harm.
